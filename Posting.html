Senior Software Developer

TiMaven, LLC is hiring a full-time Senior Software Developer position.

Job Duties:

1. Analyze information and user needs to conceive, design, implement and troubleshoot Web Applications.

2. Confer with data processing and project managers to build-out complimentary Mobile Applications including Android and IOS from concept to final implementation.

3. Determine feasibility of new and proposed designs within time and cost constraints.

4. Utilize data management and big data for various analyses across departments and metrics.

5. Obtain and evaluate information on factors such as reporting formats required, costs, or security needs to determine hardware configurations.

6. Manage the IT infrastructure from testing, building and deployment to release and operation.

7. Monitor and analyze website and mobile app’s load and be ready with infrastructure able to scale up the apps according to requirements.

8. Oversee work of, and confer with, Senior Software Developer, Junior Software Developer, Data analyst.

Job Requirements:

Master’s Degree in Computer Science or related

36 months as Software Developer, Software Engineer or related position.

Deep knowledge of web development, Spring, microservice architecture.

Expertise with Java, JSP, JS, HTML, CSS, JQuery, Hibernate, Databases (Postgesql), Version control, and Docker.

Extensive experience with Java and Android App Design and Implementation, Agile and professional software development cycle (API design, version control(Git), peer code review, unit testing).
